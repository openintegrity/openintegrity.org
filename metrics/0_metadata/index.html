Client a software project that is executive on a end user device, excepted operating systems
Server a software project that can be installed on a server and is accessed remotely by client software, excepted operating systems
Library a software project used in other projects
Service a software project accessible remotely (but cannot be installed on a server) Tor includes directory services that could in theory be redeployed and therefore the Tor project doesn't include a Service component according to this definition
Platform Package Managers, Operating systems, devices, hardware, networks. Platforms will be included in the data gathering effort in order to clarify the full dependency of software projects, however they won't be the subject of specific efforts of collection.
