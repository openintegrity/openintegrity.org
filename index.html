The Open Integrity Initiative is booting up.

We aim to provide reliable, consistent and evidence based answers to questions such as:

  • Which tools are open-source?
  • Which provide end-to-end encryption?
  • Which implement forward secrecy
  • Which support two-factor authentication?
  • Which can be downloaded securely and verified to be authentic?

You can read more about the project.

How can you get involved?

If you're collecting data or doing research on security and privacy then we'd like to hear about your work! Here are some of the measurements we're interested in:

  • Project level metrics and metadata.
  • Package and Libraries metrics (for instance around dependencies)
  • Infrastructure metrics
  • Documentation metrics
  • Localisation metrics
  • Usability metrics
  • User review metrics
  • Legal metrics
  • Human Rights Policies and Practices metrics (Ranking Digital Rights)
  • Security audit metrics
  • Net Neutrality
